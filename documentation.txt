5. Known Deployment Pitfall You Hit

If zip is created from a flat file list, folder structure is lost.

node/
python/
scripts/
6. Correct Packaging Method (Windows)

Create temp folder that preserves backend directory tree.
Remove caches from temp.
Zip the temp backend folder, not individual files.

11. Important Limits

Firebase-hosted web requires HTTPS backend due to browser mixed-content policy.
Cloudflare quick tunnel URL can change after restart.
If tunnel URL changes, rebuild web with the new URL.
